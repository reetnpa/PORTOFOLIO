# PORTOFOLIO
Repository ini berisi kumpulan portofolio proyek yang menampilkan kemampuan, pengalaman, dan pengembangan di bidang Data Analytics, Artificial Intelligence-Machine Learning (Nexus AI). Untuk mengeksplorasi berbagai hasil analisis, model AI, eksperimen, serta implementasi proyek yang telah dikembangkan secara terstruktur dan berkelanjutan.
